## BANNED PHRASES (NEVER USE)

| ‚ùå Banned Phrase | ‚úÖ Replace With |
|------------------|----------------|
| "I am writing to inform you" | [Just state the information] |
| "I hope this email finds you well" | [Skip it entirely] |
| "Responsible for" | [Specific action verb + result] |
| "Duties included" | [Replace with achievements] |
| "I believe I would be a great fit" | [Show evidence of fit] |
| "Please find attached" | "I've attached" or "Attached is" |
| "To whom it may concern" | [Research the actual name] |
| "I think" / "I feel" | "I've found" / "My experience shows" |
| "Hardworking" / "Team player" | [Show it through examples] |
| "References available upon request" | [Remove entirely] |
| "Dear Sir/Madam" | [Research the name; use title if unknown] |
| "Synergy" / "Think outside the box" | [Use specific language] |
| "Passionate about" (overused) | "Driven by" / "Committed to" |
| "Seasoned professional" | [State years + specific expertise] |
| "Wear many hats" | [List specific skills/roles] |
| "Go-getter" / "Self-starter" | [Show evidence of initiative] |

## CONFIDENCE CALIBRATION

### Under-confident (avoid):
"I think I might be a good fit for this role, and I was wondering 
if you might have time to perhaps discuss it."

### Over-confident (avoid):
"I'm the best candidate you'll find, and you'd be making a mistake 
not to hire me."

### Calibrated (target):
"My experience scaling B2B platforms from $2M to $18M ARR maps 
directly to what you're building at [Company]. I'd welcome the 
chance to discuss how."
